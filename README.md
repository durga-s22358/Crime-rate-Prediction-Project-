# 🚔 Crime Rate Prediction Using Machine Learning (LSTM)

## 📌 Project Overview

The **Crime Rate Prediction Project** is a machine learning-based application designed to analyze historical crime data and predict future crime trends across different regions. This project leverages advanced deep learning techniques, particularly **Long Short-Term Memory (LSTM)** networks, to identify patterns and forecast crime rates with improved accuracy.

The system helps in understanding crime patterns over time, enabling better decision-making for law enforcement agencies, government authorities, and researchers.

---

## 🎯 Objectives

* Analyze historical crime data by region and year
* Predict future crime rates using time-series forecasting
* Visualize crime trends through graphs and dashboards
* Improve accuracy using feature engineering and deep learning models

---

## 🧠 Technologies Used

* **Programming Language:** Python
* **Libraries & Frameworks:** Pandas, NumPy, Matplotlib, Scikit-learn, TensorFlow/Keras
* **Model:** LSTM (Long Short-Term Memory)
* **Database:** CSV / SQLite
* **Visualization:** Matplotlib / Power BI / Tableau
* **GUI (Optional):** Tkinter / Flask

---

## 📊 Dataset Features

The dataset includes crime records with the following attributes:

* State/UT
* Year
* Murder
* Attempt to Murder
* Rape (Custodial & Other)
* Kidnapping & Abduction
* Robbery, Theft, Burglary
* Total IPC Crimes

---

## ⚙️ System Workflow

1. **Data Collection & Preprocessing**

   * Clean and normalize crime dataset
   * Handle missing values
   * Feature scaling

2. **Model Training**

   * Build LSTM model for time-series prediction
   * Train using historical crime data
   * Evaluate using RMSE and MAE

3. **Prediction**

   * Forecast future crime rates for selected regions
   * Generate insights based on trends

4. **Visualization**

   * Graphs showing past vs predicted crime rates
   * State-wise analysis
